# No Fue Más - Landing Page

Página web promocional para el álbum **No Fue Más** de **Chuflay**.

El sitio presenta la portada del álbum, una sección de previews de canciones, enlaces oficiales a redes sociales y una cuenta regresiva para el lanzamiento.

## Contenido

* Portada principal del álbum
* Reproductor personalizado para previews
* Lista de canciones
* Enlaces oficiales de Chuflay
* Diseño responsive para celular, tablet y escritorio
* Estilo visual oscuro con detalles dorados
